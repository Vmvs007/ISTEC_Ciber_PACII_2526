# ISTEC_Ciber_PACII_2526-
Repositório com exemplos, exercícios e materiais da unidade curricular Programação Aplicada à Cibersegurança II (PAC II). Os conteúdos focam o desenvolvimento em Python, abordando funções, manipulação de ficheiros JSON e Programação Orientada a Objetos (classes, objetos, métodos, encapsulamento, herança, polimorfismo e override).
